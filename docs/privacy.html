개인정보 처리방침

1. 데이터 수집 금지 원칙

C'AGOK은 사용자가 입력한 어떠한 금융 정보, 거래 내역, 개인 식별 정보를 외부 서버로 전송하거나 수집하지 않습니다.

앱 실행 시 발생하는 모든 트래픽은 오직 앱 업데이트 확인 및 사용자가 직접 설정한 클라우드 백업 시에만 발생합니다.

2. 데이터 저장 방식

모든 가계부 데이터는 사용자의 로컬 디스크 내 SQLite 데이터베이스 파일 에 저장됩니다. 해당 데이터에 대한 관리 책임은 전적으로 사용자에게 있으며, 앱 삭제 시 로컬 데이터도 함께 삭제될 수 있으므로 주의가 필요합니다.

3. 제3자 제공 및 위탁

본 서비스는 별도의 회원 가입 절차가 없으며, 사용자의 데이터를 제3자에게 제공하거나 처리를 위탁하는 과정이 존재하지 않습니다. OneDrive 동기화 기능을 사용할 경우, 해당 데이터는 사용자의 개인 OneDrive 계정에만 저장됩니다.

안전한 로컬 가계부

C'AGOK은 사용자의 프라이버시를 최우선으로 생각합니다.
당신의 자산 정보는 오직 당신의 기기에만 머뭅니다.
